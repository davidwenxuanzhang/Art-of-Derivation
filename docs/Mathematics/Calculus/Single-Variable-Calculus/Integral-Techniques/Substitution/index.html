Substitution

Substitution is the most common integral technique in elementary calculus. Recall what we mentioned in common integrals that we use uuu and du\mathrm{d}udu instead of xxx and dx\mathrm{d}xdx, and this is because uuu may represent an entire function, not just a single variable. We will tackle with this more specifically in this section.

The Substitution Rule​

We will start with this integral question:

∫3x2sin⁡x3 dx= ?\int{3x^2\sin{x^3}}\ \mathrm{d}x=\ ?∫3x2sinx3 dx= ?

Note that there is an product, which is beyond all common integrals, so we need to simplify to an easier form. An example is below:

Solution

Let: u=x3u=x^3u=x3

dudx=3x2du=3x2 dx\begin{align*} \frac{\mathrm{d}u}{\mathrm{d}x}&=3x^2\\ \color{red}{\mathrm{d}u}&=\color{red}{3x^2\ \mathrm{d}x}\\ \end{align*}dxdu​du​=3x2=3x2 dx​

Plug in back to the question:

∫3x2sin⁡x3 dx=∫sin⁡u du=∫sin⁡u du=−cos⁡u+C=−cos⁡x3+C\begin{align*} \int{\color{red}{3x^2}}\sin{x^3}\ \color{red}{\mathrm{d}x}&=\int{\sin{u}}\ \color{red}{\mathrm{d}u}\\ &=\int{\sin{u}}\ \mathrm{d}u\\ &=-\cos{u}+C\\ &=-\cos{x^3}+C \end{align*}∫3x2sinx3 dx​=∫sinu du=∫sinu du=−cosu+C=−cosx3+C​

Hence:

∫3x2sin⁡x3 dx=−cos⁡x3+C■\int{3x^2\sin{x^3}}\ \mathrm{d}x=-\cos{x^3}+C \tag*{$\blacksquare$}∫3x2sinx3 dx=−cosx3+C■

By replacing the variable xxx into a function of uuu, we can simplify the overall question to a common integral. As you can see, this is a method we inversely use the Chain Rule, and generally, it is called:

The substitution rule

Given a differenciable function u=g(x)u=g(x)u=g(x), and fff is continuous on its interval, then:

∫f(g(x))g′(x) dx=∫f(u) du\int{f\left(g\left(x\right)\right)g'(x)}\ \mathrm{d}x=\int{f\left(u\right)}\ \mathrm{d}u∫f(g(x))g′(x) dx=∫f(u) du
Definite Integral

In the case of Definite Integral, the bound should also changes, as:

∫abf(g(x))g′(x) dx=∫g(a)g(b)f(u) du\int_{a}^{b}{f\left(g\left(x\right)\right)g'(x)}\ \mathrm{d}x=\int_{g(a)}^{g(b)}{f\left(u\right)}\ \mathrm{d}u∫ab​f(g(x))g′(x) dx=∫g(a)g(b)​f(u) du

Direct Substitution​

Direct Substitution is a method created by the author himself. It simplify the process of substitution without write out uuu and du\mathrm{d}udu. I will show this with an example:

∫3x2sin⁡x3 dx= ?\int{3x^2\sin{x^3}}\ \mathrm{d}x=\ ?∫3x2sinx3 dx= ?

This time, let's try what I called Direct Substituton:

Direct Substitution Approach

After analyze, we want to substitute x3x^3x3. All we need to do is substitude dx\mathrm{d}xdx to a fraction where the top is dx3\mathrm{d}x^3dx3 and the bottom is the derivative of x3x^3x3 based on xxx, as:

∫3x2sin⁡x3 dx=∫3x2sin⁡x3 d(x3)3x2=∫sin⁡x3 d(x3)\int{3x^2\sin{x^3}}\ \mathrm{d}x=\int{\cancel{3x^2}\sin{x^3}}\ \frac{\mathrm{d}(x^3)}{\cancel{3x^2}}=\int{\sin{x^3}}\ \mathrm{d}(x^3)∫3x2sinx3 dx=∫3x2sinx3 3x2d(x3)​=∫sinx3 d(x3)

And all we need is to consider x3x^3x3 as uuu and integrate normally:

  ⟹  ∫sin⁡x3 d(x3)=−cos⁡x3+C\implies \int{\sin{x^3}}\ \mathrm{d}(x^3)=-\cos{x^3}+C⟹∫sinx3 d(x3)=−cosx3+C

More importantly, for definite integral, it doesn't need to change bounds, as:

∫ab3x2sin⁡x3 dx=∫absin⁡x3 d(x3)=−cos⁡x3∣ab=−cos⁡b3+cos⁡a3\int_{a}^{b}3x^2\sin{x^3}\ \mathrm{d}x=\int_{a}^{b}{\sin{x^3}}\ \mathrm{d}(x^3)=-\cos{x^3}\big|_{a}^{b}=-\cos{b^3}+\cos{a^3}∫ab​3x2sinx3 dx=∫ab​sinx3 d(x3)=−cosx3​ab​=−cosb3+cosa3
