# Integral with $\sqrt{x^2-a^2}$

### 45
$$
\int{\frac{1}{\sqrt{x^2-a^2}}}\ \mathrm{d}x=\frac{x}{\left|x\right|}\operatorname{arcosh}
{\frac{\left|x \right|}{a}}+C=\ln{\left|x+\sqrt{x^2-a^2}\right|}+C
$$
