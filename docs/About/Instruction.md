# Instruction

## Usage Recommendation

- All documents are organized in a logical sequence; following the given order is strongly recommended. 
- Before viewing the full derivation through the collapsable bar, attempt to work through each step independently using pen and paper.
- Detailed solutions are provided in the collapsible sections for verification and deeper understanding.
- Maintaining a dedicated notebook for your own derivations is highly encouraged.
